# mark_pattern

소프트웨어 설계·아키텍처를 다루는 Claude Code 플러그인.

## 상태

작업 중. 현재는 저장소 골격만 있고 플러그인 구성 요소는 아직 없다.

## 목표

기존 스킬들이 대부분 **사후적**(버그를 맞은 뒤에 발동)이라는 관찰에서 출발한다.
이 플러그인은 **코드를 쓰기 전에** 발동하는 설계 단계 스킬을 목표로 한다.

## 구조 (예정)

```
.claude-plugin/
  plugin.json      플러그인 매니페스트
skills/            설계 단계 스킬
commands/          슬래시 커맨드
agents/            서브에이전트
hooks/             결정론적 검사
```

## 설치

아직 배포 가능한 상태가 아니다.
